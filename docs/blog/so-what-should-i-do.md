# 那我该怎么办？

**作者：Michael Chomsky（[@michael_chomsky](https://x.com/michael_chomsky)）**
**发布时间：2026年2月11日**
**原文：[So what should I do about it](https://x.com/michael_chomsky/status/2021546591018983441)**

---

Matt Schumer 写了一篇叫《大事正在发生》的文章。如果你还没读过，我推荐你去看看。核心观点是：AI 正在以惊人的速度进步，并且刚刚跨过了足以产生颠覆性影响的门槛。

而这很可能是 AI 有史以来最弱的时候。
那你该怎么办？

以下是一些具体步骤，确保你在利用这场变革，而不是成为它的牺牲品。

## 1. 永远使用最强的模型

地球上大多数人根本不知道模型已经变得多么强大，因为他们从未尝试过最新一批模型，比如 Codex 5.3 High、Opus 4.6 或 ChatGPT Pro。这些模型的智能程度难以想象。我们集体习惯了 AI 的平庸，以至于很难想象它在比你预期更多的任务上超越你。

我写作多年，多年来 AI 对我的写作弊大于利，拖慢速度、降低清晰度。但到了最新一批模型，我终于看到它们写出了比我自己更好的东西——这颗药很难咽下去。

我的第一反应是否认。也许文字没那么好？也许这个商业计划其实不如我的？

但最终我接受了：是的，这些模型现在已经足够聪明，能像一个经验丰富的写手、开发者或任何专业人士站在你身后提建议一样，贡献同等甚至更多的价值。

## 2. 把最难的任务交给 AI

我生活中有很多事一直在拖延，仅仅因为我知道它们需要超常的努力。

三件事浮现在脑海：

- 弄清楚如何改善我的右髋——手术效果不好，困扰了我多年，看了很多医生和理疗师都没用。
- 一个代码仓库里的大规模迁移，我已经花了一百多个小时，用上一代工具还需要好几天。
- 一个我想了好几年的项目，一直没时间做，而且我知道它不仅需要大量时间投入，还需要我目前不具备的对项目范围的清晰认知。

所有这些问题，如果投入足够的算力和最聪明的模型，大概都能快得多。我和实现这些目标之间唯一的障碍，可能只是鼓起勇气去问，如果第一次失败了，就投入更多算力。这也是我发完这篇文章后要做的事。

## 3. 花更多算力

世界上几乎所有人都没有花足够的算力来最大化投资回报率。智能现在极其便宜，而世界上有足够多的机会可以把算力套利成财务或其他回报。

如果你一次只跑一个 agent 处理一个难题，问问自己：为什么不跑几十个 agent？为什么不同时用多个模型？为什么不从多个角度攻击问题，同时做更多任务？

我们还没有建成有效扩展算力的系统或基础设施，但这本身就是另一个套利机会。如果你比别人花更多算力，你很可能会赢。

## 4. 多做实验

去折腾。试试 OpenClaw 这样的项目。下载 Claude Code 或 Claude Desktop。买 200 美元的 Anthropic 订阅，再买 200 美元的 OpenAI 订阅。然后真正在日常工作中狠狠地用它们。设定一个月度「算力预算」然后花掉它。

智能很便宜，你的学习速度才是制胜关键。如果你有员工，给他们每人 400 美元用于 LLM。如果这没有改善他们的产出，要么是工作流有问题，要么是工作还不够清晰以至于无法委派——当然也有例外。

受益最多的是那些实验更多、更多次出手的人。

我这周推出了 SetupClaw，一个帮忙碌的创始人和高管通过一条推文就能搭建 OpenClaw 的服务。没有网站，没有产品。我只是尝试了一件新事，就得到了回报——第一周就有 1.5 万美元收入和数千新粉丝。

先发优势很大，先行者最轻松。他们不需要清晰的认知、对世界的深刻理解，甚至不需要能力。他们只需要做第一个。

## 5. 现在就赚大钱（如果这是你在意的事）

当你能这么快行动时，有很多钱可以赚。LLM 可以帮你构建项目、制定商业计划、做客服、帮你迭代、发现你判断中的漏洞，以及白领工人能做的大多数事情。

企业可以更精简，行动更快。历史上从来没有比现在更容易赚到一大笔钱的时候了，任何有合理智力、以此为目标并投入足够时间的人，都能让自己进入改变命运的射程范围。

这是因为财富来自不断尝试。历史上，普通人可能需要几十年才能尝试并成功创业，因为每次下注都会吞噬数年时间。现在你可以更快地下注、失败、学习。

## 6. 赌自己

如果历史上确实从未有过这么多机会，那唯一合乎逻辑的做法就是去做那些天花板极高的事情。

一份没有股权的工作天花板极低，无论薪资多高。机会成本也是巨大的——你每周花 40 多个小时，可能没有实现目标所需的完全自由。

但赌自己、创业，天花板要高得多——取决于做什么生意。把自己放在极端结果有可能发生的位置，行不通就果断砍掉。机会成本实在太高了。

## 7. 赌聪明人

聪明人将从这场变革中获益最多，因为他们对存在的机会有足够好的理解，能有效利用它们。有了合适的工具，他们将变得无限强大，而异类将变得更加极端。我预计收入不平等会严重加剧——利用者在利用，利用变得更精简，利用的回报更加集中。

## 8. 赌有毅力和强烈自信的人

既然机会的天花板现在高了这么多，那些相信自己、面对障碍不放弃的人就有了不可思议的优势。要说出「我要治愈某种癌症」「我要在 5 年内成为亿万富翁」「我要登月」，需要一种特殊的人。

历史上，这种人几乎总是失败或达到平庸的结果。但现在，他们完全有可能成功，因为他们选择了一条天花板极高的路，而且拥有比历史上任何人都多的工具去抵达那里。

## 9. 做杠杆化的赌注

我不是在说金融市场上的杠杆。我是说，既然任何具备合理毅力、态势感知力、实验舒适度和智力的人成功的概率如此之高，而你自己也具备这些特质，那就应该把自己杠杆化到极致。

但要确保下行是可承受的。做不对称赌注：大的上行空间，可控的风险，这样你就能持续出手。这不是财务建议。我的意思是优化速度和出手次数，但不要把自己炸掉。

关键是：任何能让你行动更快、出手更多、因此学习更快的事情，在你使用了足够算力的前提下，大概都是值得的。

## 10. 别焦虑

最后一点主要是写给我自己的。生活在一个机会如此之多的时代，我深受极度焦虑之苦，这很不健康，对实现目标毫无帮助。

如果我满怀近乎妄想的自信，我的成功概率会更高，焦虑也会消失。

我个人的做法是深呼吸几次，冷静下来，弄清楚应该把精力用在哪里以最大化结果。因为虽然算力几乎无限，但我引导算力的注意力和能力是有限的。

最终胜出的人，是那些没有被这个新现实击垮、无情地砍掉众多小机会、去追逐大海里最大那条鱼的人。
