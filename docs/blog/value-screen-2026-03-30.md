# 每日价值筛选 2026-03-30：AI 基础设施里最便宜的三张票

今天这版我不想装模作样地扫一堆票。**真正值得看的，就是 AI 基础设施和数字分发链条里，估值还没被抬上天、而且已经能出真钱的公司。**

结论先放前面：**今天最值得盯的三张票是百度、联想集团、中国移动。**

它们有个共同点：都不是市场最性感的名字，但都站在 AI 落地的真钱环节里——云、端侧设备、网络与数据中心——而且估值都还没疯。

## 今日筛选结论

### 🇺🇸 百度 Baidu (BIDU) — 108.12 USD

**为什么被低估：** 市场还把百度当成“老搜索”，但它真正该看的，是 AI 云、文心生态和自动驾驶期权。
**关联主题：** AI 应用层最终会向云基础设施、模型调用和流量入口回流。

| 指标 | 数值 |
|------|------|
| 市值 | 367.9 亿美元 |
| TTM 营收 | 1290.8 亿元人民币 |
| 盈利状态 | ✅ |
| Forward P/E | 11.6x |
| 营收增速 | -4.1% |
| FCF | 167.9 亿元人民币 |
| 分析师目标价 | 175.6 USD |

**催化剂：**
- 文心模型商业化继续落地，带动云业务重新加速
- 自动驾驶与 Robotaxi 如果继续扩大运营区域，市场会重新给期权定价
- 当前 PB 约 0.95x，意味着市场对资产质量极不信任，只要基本面不再恶化，估值修复空间就不小

**风险：**
- 广告业务恢复不够快
- AI 投入高、利润兑现慢
- 中国互联网平台整体估值折价短期难消

### 🇭🇰 联想集团 (0992.HK) — 9.40 HKD

**为什么被低估：** 它不是“AI 讲故事公司”，它是最接近 AI 终端放量现实的一家硬件平台，估值却只有单数 PE。
**关联主题：** AI 先在数据中心烧钱，随后一定下沉到 PC、边缘设备和企业终端，联想正卡在这条链上。

| 指标 | 数值 |
|------|------|
| 市值 | 1166.0 亿港元 |
| TTM 营收 | 784.7 亿美元 |
| 盈利状态 | ✅ |
| Forward P/E | 8.4x |
| 营收增速 | 18.1% |
| FCF | 7.5 亿美元 |
| 分析师目标价 | 12.23 HKD |

**催化剂：**
- AI PC 渗透率抬升，PC 周期从换机转向结构升级
- 服务器与基础设施业务如果继续改善，市场会把它从“低毛利 PC 厂”重估为“端+云一体化设备商”
- 单位数到低双位数 PE，在港股 AI 相关资产里已经便宜得有点离谱

**风险：**
- PC 行业天然周期性强
- 硬件业务利润率薄，任何需求回落都会放大利润波动
- 如果 AI PC 只是营销词而不是刚需，估值修复会拖慢

### 🇭🇰 中国移动 (0941.HK) — 78.05 HKD

**为什么被低估：** 市场把它当成慢电信，但它其实是中国最稳的数据中心、云网融合和算力承载平台之一。
**关联主题：** AI 不只要芯片，还要网络、IDC、云资源和大客户交付，中国移动吃的是“卖水”逻辑。

| 指标 | 数值 |
|------|------|
| 市值 | 1.71 万亿港元 |
| TTM 营收 | 1.05 万亿元人民币 |
| 盈利状态 | ✅ |
| Forward P/E | 9.7x |
| 营收增速 | 2.5% |
| FCF | 765.6 亿元人民币 |
| 分析师目标价 | 98.54 HKD |

**催化剂：**
- 算力网络、IDC、政企业务继续扩张，提升“公用事业估值”向“数字基础设施估值”的切换概率
- 高股息本身就是安全垫，市场风险偏好回落时反而更容易获得资金回流
- 如果 AI 资本开支进入长期化阶段，中国移动这种底层承载资产会越来越值钱

**风险：**
- 增长不快，重估需要时间
- 国资大盘股天然容易被市场低估
- 监管与资本开支节奏可能压制短期利润弹性
