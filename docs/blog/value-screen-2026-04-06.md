# 每日价值筛选 2026-04-06：算力硬件、光学复苏、卖水网络里的三张便宜票

今天这版的核心就一句话：**别追最热，去找 AI 链条里还没被市场狠狠干贵的真钱资产。**

我今天只留三张票：**戴尔科技、舜宇光学科技、中国移动。**

它们分别卡在三个位置：

- **企业算力硬件**：谁在给 AI 服务器和企业基础设施卖铲子
- **端侧与车载光学**：谁在吃 AI 手机、智能汽车、视觉升级的量价修复
- **网络与数据中心**：谁在长期收“AI 卖水钱”

这三张票的共同点很简单：**估值没疯、业务不是空气、现金流还在。**

## 今日筛选结论

### 🇺🇸 戴尔科技 Dell Technologies (DELL) — 174.37 USD

**为什么被低估：** 市场知道它能卖服务器，但还没完全按“AI 基础设施分销平台”给它估值。
**关联主题：** 企业 AI 落地第一阶段，不是人人都自研模型，而是先买服务器、存储、网络和集成方案。

| 指标 | 数值 |
|------|------|
| 市值 | 1155.5 亿美元 |
| TTM 营收 | 1135.4 亿美元 |
| 盈利状态 | ✅ |
| Forward P/E | 12.0x |
| 营收增速 | 39.5% |
| FCF | 66.1 亿美元 |
| 分析师目标价 | 170.54 USD |

**催化剂：**
- AI 服务器订单继续兑现，市场会把它从“传统 PC/服务器厂”重新定价为“AI 基础设施渠道核心”
- 企业本地部署需求如果继续增强，戴尔这种集成能力会比纯硬件标签更值钱
- 12 倍左右 forward PE，放在接近 40% 的营收增速面前，不算贵，甚至有点保守

**风险：**
- 行业竞争激烈，硬件业务天然毛利率不厚
- 如果 AI 服务器需求短期波动，情绪会先杀估值
- 当前股价已反映一部分预期，短线弹性未必最大

### 🇭🇰 舜宇光学科技 (2382.HK) — 59.90 HKD

**为什么被低估：** 市场还把它当手机光学周期股，但它真正值钱的，是高端镜头、车载光学和视觉升级的复合杠杆。
**关联主题：** AI 手机、机器人、智能驾驶都离不开更高规格的摄像头和感知模组，最后受益的不是嘴上喊 AI 最凶的人，而是能稳定出货的光学供应商。

| 指标 | 数值 |
|------|------|
| 市值 | 642.8 亿港元 |
| TTM 营收 | 432.3 亿元人民币 |
| 盈利状态 | ✅ |
| Forward P/E | 12.0x |
| 营收增速 | 21.3% |
| FCF | 11.9 亿元人民币 |
| 分析师目标价 | 80.92 HKD |

**催化剂：**
- 智能手机影像规格继续上行，ASP 有修复空间
- 车载镜头与 ADAS 业务如果继续放量，市场会开始按“汽车视觉零部件”而不是“手机零件”来估它
- 目标价相比现价还有明显空间，而 12 倍左右 forward PE 仍处在可以接受的价值区间

**风险：**
- 手机链恢复如果不及预期，盈利弹性会被拖累
- 消费电子周期反复，估值修复节奏容易被打断
- 光学制造业技术迭代快，产品 mix 一旦掉队，利润率会很难看

### 🇭🇰 中国移动 (0941.HK) — 80.05 HKD

**为什么被低估：** 市场总把它当低增长电信股，但它其实是中国最稳的一类云网基础设施和数据中心承载资产。
**关联主题：** AI 真正大规模落地，不只要芯片，更要网络、IDC、云资源和企业交付能力；中国移动吃的是最底层、最稳定的“卖水钱”。

| 指标 | 数值 |
|------|------|
| 市值 | 1.75 万亿港元 |
| TTM 营收 | 1.05 万亿元人民币 |
| 盈利状态 | ✅ |
| Forward P/E | 10.0x |
| 营收增速 | 2.5% |
| FCF | 765.6 亿元人民币 |
| 分析师目标价 | 98.36 HKD |

**催化剂：**
- 算力网络、政企业务、IDC 和云资源继续扩张，估值逻辑会从“电信公用事业”慢慢切到“数字基础设施”
- 超高股息本身就是安全垫，风险偏好回落时反而更容易被资金重新拥抱
- 如果市场开始重新偏好现金流和分红，中国移动这种票会显得越来越顺眼

**风险：**
- 增长确实不快，重估往往慢半拍
- 国资大盘股容易长期被低估，便宜能便宜很久
- 资本开支节奏与监管方向会压制短期利润弹性

## 今天怎么排位

如果只选一个，我会先看 **舜宇光学**。

原因不复杂：**增长、估值、预期差** 这三件事，它今天的组合最漂亮。

- 想要**进攻性更强**：看舜宇光学
- 想要**吃企业 AI 开支**：看戴尔
- 想要**最稳、能扛、还能收息**：看中国移动

说白了，这三张票都比“继续追已经被讲烂的热门 AI 龙头”更像正常人能拿得住的标的。
